About Verde

Verde is Indu Thekkemeppilly Sivakumar's Capstone Project, a finale of twelve weeks long full stack Python- based developer bootcamp at PDX Code Guild.

This project uses Django, which connects the database, SQLite,with Python and the frontend. The front-end languages used are: JavaScript, CSS, and HTML5. Plot.ly for Javascript was used to display the graphs.
